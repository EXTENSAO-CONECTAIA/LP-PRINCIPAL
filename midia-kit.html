Stack Visual

Tecnologias e recursos utilizados na construção da landing page.

Inter (Google Fonts)

Fonte principal com pesos 400–900. Preconnect + preload para carregamento rápido.

CSS Custom Properties

Variáveis CSS (--primary, --violet, --gray-*) para consistência de cores em todo o sistema.

SVG Inline

Ícones de IAs (ChatGPT, Claude, Gemini, Perplexity) e ícones de interface em SVG inline para performance.

CSS Animations

Animações de float, pulse, fade-in e contadores para engajamento visual.

Intersection Observer

Cards aparecem com animação de slide-up conforme o usuário rola a página.

Schema.org / JSON-LD

Dados estruturados para SEO: SoftwareApplication + FAQPage para rich snippets no Google.
